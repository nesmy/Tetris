### Installation

<!-- 1. Get a free API Key at [https://example.com](https://example.com) -->
#### Web

1. Clone the repo
   ```sh
   git clone https://github.com/nesmy/Tetris.git
   ```
2. Build the project Debug Release
   ```sh
   ./buildweb.sh
   ```
3. Bin
   ```sh
   html in build/bin
   ```

#### Linux*

1. Clone the repo
   ```sh
   git clone https://github.com/nesmy/Tetris.git
   ```
2. Build the project Debug Release
   ```sh
   ./build.sh
   ```
3. Run
   ```sh
   ./run.sh
   ```
